## Serverless - Lambda Function

### Serverless
- Sem gerenciamento de servidores
- Recursos (servidor, infraestrutura) gerenciados pelo provedor
- Escalonável
- Atenção para a aplicação ao invés da infraestrutura
- Serviços oferecidos pela AWS serverless:
  - Lambda Functions
  - AWS API Gateway
  - Dynamo DB
  - Amazon S3
  - Aws Aurora Serverless

### Lambda function
- Serverless
- Executa somente quando requisitado (Triggers)
- Escalonamento automático
- Limite de tempo - 15min por requisição
- Integração com várias linguagens de programação
- Integração com vários serviços da AWS
> [!IMPORTANT]
> Cobrado por requisição
